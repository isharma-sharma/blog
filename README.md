# front end react redux



```
1.used react-router for showing different pages on the browser
2.used redux-promises which act as a middleware between  action-creator and reducer
3.used redux-form for creating the form field for blog
4.used axios for api call
5.added validation to the input field of the form
```


 # dependencies
 ```
 > npm install axios
 > npm install redux-promises
 > npm install router
 > npm install redux-form
```
